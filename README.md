# Jogo de Adivinhação em Python

Aplicação em Python executada via terminal onde o usuário deve adivinhar um número secreto gerado aleatoriamente.

## Funcionalidades

- Seleção de nível de dificuldade
- Geração de número aleatório
- Limite de tentativas por dificuldade
- Validação de entradas inválidas
- Feedback de tentativas (maior/menor)
- Histórico de tentativas
- Opção de jogar novamente

## Conceitos aplicados

- Loops `while`
- Condicionais
- Listas
- Módulo `random`
- Controle de fluxo
- Validação de entrada de dados

## Execução

```bash
python jogo_adivinhacao.py
```

## Objetivo

Praticar lógica de programação, controle de fluxo e interação com o usuário via terminal.
